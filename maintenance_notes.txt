
# Maintenance Notes for FarseerXenuine.github.io

## General Information
- This project is a website for Farseer Xenuine, an anti-cheat service.
- The project uses HTML, CSS, and JavaScript.
- The CSS files are organized into `style.css` for general styles and `mobile.css` for mobile-specific styles.

## CSS Media Queries
- Ensure that media queries are used to handle different screen sizes.
- Example:
  ```css
  @media screen and (max-width: 767px) {
    /* Mobile-specific styles */
  }
  ```
